## 📌 프로젝트 개요

SER1ES AI 플랫폼 학습용 데이터셋 30개(6개 프로덕트 × 5개 퀄리티 티어)의 가격/물량 리밸런싱.

**목표**: STARTER는 "안 끼느니만 못 하다"가 아닌 "안 끼는 것보단 낫다" 수준으로 최소화. ULTIMATE는 "VC가 투자하기도 전에 돈이 필요한" 수준으로 극대화.

## 💰 매출 시뮬레이션 (돈미새 모드)

### 가격 스프레드 (1프로덕트 기준)
```
STARTER:    0 CR        ← 무료. 미끼. 마약의 첫 맛.
BASIC:     30~45 CR     ← 커피 한 잔 값. "이 정도는..."
ADVANCED: 700~1,000 CR  ← 진지해지기 시작. 지갑이 떨림.
PREMIUM: 2,800~3,500 CR ← 팀장 결재 필요. 회의 소집.
ULTIMATE: 7,000~10,000 CR ← C레벨 결재. 이사회 보고. 근데 결국 삼.
```

### BEFORE 최대 매출 (6프로덕트 ULTIMATE 전부 구매)
```
1,500 + 2,000 + 1,800 + 1,500 + 1,800 + 2,000 = 10,600 CR
```

### AFTER 최대 매출 (6프로덕트 ULTIMATE 전부 구매)
```
8,000 + 10,000 + 9,000 + 7,000 + 9,500 + 10,000 = 53,500 CR
```

### 🔥 매출 증가: **+404.7%** 🔥

> 한 명의 엔터프라이즈 고객이 6개 프로덕트 ULTIMATE 전부 사면
> 53,500 CR × $0.0025(enterprise rate) = **$133.75/건**
> 이건 데이터셋 접근비만임. 트레이닝 GPU 비용은 별도. ㅋㅋ

## 📊 티어별 설계 원칙

| 티어 | 네이밍 | 설계 의도 |

| BASIC | "Lite" (라이트) | 최소한의 과금. 결제 습관 형성. 심리적 장벽 제거. |
| ADVANCED | "Arsenal" (무기고) | 프로덕션 진출 단계. 진짜 돈 쓰기 시작. ROI 보이기 시작. |
| PREMIUM | "Overlord" (지배자) | 엔터프라이즈 전용. 예산 있는 팀만 접근. 경쟁사와 차별화. |
| ULTIMATE | "God" (신) | 이것보다 좋은 건 없음. FOMO 극대화. "남들은 이거 쓰는데?" |

## 🛠️ 기술 변경 사항

### 수정 파일:
1. **config.py** — `PRODUCT_DATASETS` 30개 항목 전체 리밸런스
2. **train.html** — `DATASET_FALLBACK` 프론트엔드 하드코딩 동기화

### 네이밍 변경:
- `Sampler` → `Crumbs/Scraps` (부스러기 느낌 극대화)
- `Baseline` → `Lite` (가벼움 강조)
- `Pro` → `Arsenal` (무기 = 도구 = 진지함)
- `Master` → `Overlord` (지배자 = 엔터프라이즈)
- `Nexus` → `God` (신 = 최종 보스 = 더 이상 위가 없음)
